# Mongo Users

A web-application with Node, Mongodb, Mongoose, Express, Handlebars & Bootstrap 

## Setup & run

- Clone the repo
- You need a running instance of mongod
- `npm i`
- `npm run dev`

## Features
*so far*

- Database seeding
- Express with Handlebars/Bootstrap
- Users overview
- Single user infos
- Search by age greater than
